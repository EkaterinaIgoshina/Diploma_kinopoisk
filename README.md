# Diploma_kinopoisk
# Автоматизация тестирования сайта Кинопоиск.

## Задача проекта.

   Автоматизация UI- и API-тестов из проекта по ручному тестированию сайта Кинопоиск.
   
## Структура проекта.

### Папки:
     TEST_UI - UI-тесты, методы для UI-тестирования;
     TEST_API - API-тесты, методы для API-тестирования; 
     requirements.txt - список необходимых библиотек и их версии.
     

## Запуск тестов.
     pytest TEST_API - запуск только API-тестов;
     pytest TEST_UI - запуск только ui- тестов;
     pytest TEST_API TEST_UI - запуск api и ui тестов;
     pytes --alluredir=./allure-result - запуск тестов с генерацией отчёта в папке allure-result;
     allure serve allure-result - создание веб-отчёта по результатам тестирования.
